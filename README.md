# web-technology-pbl
This repository contains banking website project made by me and my classmates during 1st year <br>
<b>OVERVIEW OF PROJECT</b> 
<br>
The Banking Website is a secure and responsive online platform that allows users to manage their bank accounts digitally. It enables features like account registration, balance checking, fund transfers, transaction history, and profile management. Built with modern web technologies, the system ensures data security, real-time processing, and a user-friendly interface for both customers and administrators.
